# trading-app v0.1.0

**Date**: 2026-06-16  
**Status**: Reference integrator app — **UAT-ready**, not Live-ready

## Summary

First public release of the app layer formerly known as `theman`. Wires three sibling packages into a Windows-deployable stack with config, storage, reporting, and param sweep tooling.

## Install

```powershell
git clone https://github.com/timhwchuang/trading-app.git
cd trading-app
python -m venv .venv
.\.venv\Scripts\activate
pip install "trading-engine @ git+https://github.com/timhwchuang/trading-engine.git@v0.2.0"
pip install "trading-backtest @ git+https://github.com/timhwchuang/trading-backtest.git@v0.1.0"
pip install "strategy-vwap-momentum @ git+https://github.com/timhwchuang/strategy-vwap-momentum.git@v0.1.0"
pip install -r requirements.txt
```

## What's included

- `trading_app_engine_ports()` — explicit Shioaji/Mock adapter + telemetry/archive/alerts
- `python -m live` — simulation/live entry (`config/config.yaml`, env secrets)
- `python -m reporting` — UAT log parser (`SIGNAL_AUDIT` / `FILL_AUDIT` / `DAILY_SUMMARY`)
- `storage/` — tick/kbar archive compatible with `trading-backtest` replay
- `sweep/` — walk-forward research (not a UAT gate)
